# HTML5 Audio Volume Demo
A minimal project demonstrating how to use HTML5 `<audio>` element with basic user interaction.

Just a project experimenting with HTML and small amounts of inline JavaScript to control audio volume using buttons, no external libraries or external plugins required.

## How to Run
1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Use the buttons to interact with the audio player

## Things to Note
- Autoplay may not work due to browser restrictions (ie. Chrome)
- Some browsers (ie. Firefox) require users to modify their browser's autoplay setting for autoplay to work
